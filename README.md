# geo-tools

地理情報を取り扱うためのコード群

## コンテンツ

- docker: 地理データを扱うためのdockerイメージを作る
- master: 地理データのマスタを作る
- streamlit: 地理データを扱うためのストリームレットアプリを作る
- map-samples: 地図描画のサンプル
- tileset: タイルセットを作る (TODO ラスター版のサンプル)
- gdal: TODO
- ui-tool: いろいろ計算する用のUIツール (TODO)
